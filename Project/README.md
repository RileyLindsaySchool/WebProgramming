# Riley Lindsay Web Portfolio

## Project Overview
This is a portfolio site describing my skills, interests and projects.

- Provides an interface for contacting me or learning about me and my skills
- Alleviates difficulties in learning important details from existing pages, such as my LinkedIn
- For potential employers or colleagues who wish to learn more about me

---

## Setup Instructions

Follow these steps to get the project running locally:

### Prerequisites
- Git
- Browser such as Microsoft Edge

### Opening Web Site
```cmd
# Clone the repository
git clone https://github.com/RileyLindsaySchool/WebProgramming.git

# Navigate into the project directory
cd WebProgramming\Project
```
Open index.html in a web browser
